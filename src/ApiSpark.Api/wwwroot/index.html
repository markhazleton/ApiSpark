About

MakeBoldSpark consolidates multiple low-volume APIs into a single, cost-effective ASP.NET Core application backed by EF Core and SQLite, deployed on Azure App Service Linux. It is part of the migration of all Spark projects to the makeboldspark.com domain and serves as the shared API layer for Make Bold Solutions and Mark Hazleton portfolio sites.
